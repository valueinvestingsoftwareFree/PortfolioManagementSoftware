What Is Money-Weighted Return?

Money-weighted return (MWR) is an investment performance metric that measures the actual return earned by an investor after considering the timing and amount of all cash flows, including deposits and withdrawals. Unlike time-weighted return, money-weighted return reflects the personal experience of the investor.

Because contributions made during strong or weak market periods can significantly influence outcomes, MWR provides a realistic picture of how well an investor's money actually performed over time.

Why Money-Weighted Return Matters

Most investors regularly contribute to their portfolios through monthly savings, retirement contributions, dividend reinvestment programs, or additional purchases. Some investors also withdraw funds periodically.

Money-weighted return incorporates all of these activities, helping investors understand the return they personally achieved rather than measuring only the effectiveness of the investment strategy itself.

Money-weighted return helps investors:
  • Measure actual investor experience.
  • Understand the impact of contributions.
  • Evaluate the timing of cash flows.
  • Assess retirement account performance.
  • Monitor long-term investing progress.
  • Review the effectiveness of savings habits.
  • Compare financial outcomes realistically.
  • Analyze dividend reinvestment effects.
  • Improve planning decisions.
  • Support goal-based investing.

For investors who actively add or remove funds from their portfolios, money-weighted return often provides one of the most meaningful measures of success.

How Is Money-Weighted Return Calculated?

Money-weighted return is commonly calculated using the Internal Rate of Return (IRR) methodology. The calculation identifies the rate of return that equates all cash inflows and outflows over the investment period.

General steps include:
  1. List all deposits into the portfolio.
  2. Record all withdrawals.
  3. Include dividend distributions if applicable.
  4. Determine the ending portfolio value.
  5. Account for the timing of every cash flow.
  6. Calculate the internal rate of return.

Although this process can become complex, portfolio software can automate calculations and provide investors with accurate results.

Money-Weighted Return vs. Time-Weighted Return

Both metrics provide valuable insights, but they answer different questions.

Money-Weighted Return

Reflects the investor's actual experience, including cash flow timing.

Time-Weighted Return

Measures the effectiveness of the investment strategy itself.

Investor Focus

Useful for evaluating personal financial progress.

Manager Evaluation

Useful when comparing investment managers objectively.

Understanding both metrics allows investors to gain a more complete perspective on portfolio performance.
