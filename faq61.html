What Is Time-Weighted Return?

Time-weighted return (TWR) is a portfolio performance metric designed to measure how an investment strategy performs independently of investor cash flows. By excluding the impact of deposits and withdrawals, it provides a clearer picture of the effectiveness of investment decisions themselves.

For investors who regularly contribute new money, reinvest dividends, or make periodic withdrawals, time-weighted return can offer a more objective assessment of portfolio performance than simply observing changes in account balances.

Why Time-Weighted Return Matters

Many investors add money to their portfolios over time or occasionally withdraw funds. These cash flows can distort traditional return calculations and make it difficult to determine whether strong results came from investment decisions or simply larger contributions.

Time-weighted return addresses this issue by separating investment performance from the timing and size of external cash flows.

Time-weighted return helps investors:
  • Evaluate the effectiveness of an investment strategy.
  • Compare performance across different periods.
  • Remove the effects of deposits and withdrawals.
  • Benchmark portfolio managers more fairly.
  • Identify long-term trends.
  • Improve performance analysis.
  • Reduce misleading interpretations.
  • Support disciplined investing decisions.
  • Compare multiple portfolios consistently.
  • Focus on true investment results.

Because of these advantages, time-weighted return is widely used by professional investment managers and performance reporting standards worldwide.

How Is Time-Weighted Return Calculated?

The basic concept behind TWR involves dividing the investment period into segments whenever external cash flows occur. The return for each segment is calculated independently and then linked together geometrically.

General steps include:
  1. Identify all cash flow dates.
  2. Divide the timeline into sub-periods.
  3. Calculate the return for each sub-period.
  4. Exclude external contributions and withdrawals.
  5. Multiply the individual period returns together.
  6. Convert the result into an overall percentage return.

Although the calculations can become complex for active investors, portfolio software can simplify the process and provide reliable performance measurements automatically.
